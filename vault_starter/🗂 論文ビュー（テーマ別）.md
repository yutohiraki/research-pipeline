# 🗂 論文ビュー（テーマ別・最近順）

> テーマごとに、各テーマ内は追加日の新しい順で並べるビュー。
> Dataview が有効なら自動で表になります。**下の「テーマ別」ブロックは自分のタグに合わせて増やす**。

---

## 🆕 最近追加された論文（全テーマ・上位20）

```dataview
TABLE WITHOUT ID
  file.link AS "論文", read_date AS "追加日",
  filter(tags, (t) => t != "literature") AS "タグ", year AS "年"
FROM "literature_notes"
WHERE status = "read"
SORT read_date DESC, year DESC
LIMIT 20
```

## 🏷️ タグ別 本数（多い順・全体像）

```dataview
TABLE WITHOUT ID
  key AS "タグ", length(rows) AS "本数"
FROM "literature_notes"
FLATTEN tags AS tag
WHERE tag != "literature"
GROUP BY tag
SORT length(rows) DESC
```

---

## 📁 テーマ別（← 自分のタグに合わせて複製・編集する）

> 使い方: 下のブロックをコピーし、見出しと `contains(tags, "＜タグ名＞")` を自分のタグに変える。
> タグは frontmatter の `tags` に入っている語（上の「タグ別 本数」で確認できる）。

### ＜テーマ名の例＞

```dataview
TABLE WITHOUT ID
  file.link AS "論文", read_date AS "追加日", year AS "年", my_rating AS "評価"
FROM "literature_notes"
WHERE contains(tags, "＜タグ名＞") OR contains(tags, "＜別のタグ＞")
SORT read_date DESC, year DESC
```
